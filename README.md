## User Stories
```
As a user, 
So that I can list a few places, 
I'd like to be able to sign up
```
```
As a user,
So that I can share a listing,
I'd like to list a new space
```
```
As a user, 
So that I can share additional listings, 
I'd like to be able to list multiple listings
```
``` 
As a user, 
So that I display information about my listing, 
I'd like to be able to name my space, provide a short description of the space, and a price per night
```
```
As a user, 
So I can share my listings availability, 
I'd like to be able to offer a range of dates for my listing
```
```
As a user, 
So I can book a listing, 
I'd like to be able to submit a request for one night
```
```
As an owner,
So that I can get a booking, 
I'd like to be able to accept a request for one night
```
```
As a owner, 
So that the date is not booked twice, 
I'd like the date that is booked to be blocked
```
```
As an user, 
So that the space is booked, 
I'd like to confirm a booking request
```
```
As a owner, 
So that my listing is filled, 
I'd like to not be blocked until the booking is confirmed
```
